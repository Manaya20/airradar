# Airadar

This project is a small scale model which depicts the actual working of the Air Traffic Control.

ATC manages hundreds of flight throughout the day at a busy airport. Therefore the following project will consisted of mall number of aircrafts and their details. The queries represented define data which can be obtained from a Air Traffic Control Room.

## Prerequisites


```bash
  pip install pandas
  pip install numpy
  pip install tkinter
```

## Working

In this Database Management Project, Information related to the Aircraft, Airline, Airport, Destination, Delay information and much more is used.
The information consists of many unique elements which would be used to retrieve desired output. To retrieve the desired output the dataset of Bengaluru airport has been taken as a csv file which was imported through Pandas and represented in GUI format by importing tkinter.

Data of all entries is stored in a csv file, which is imported and read by the module, pandas. A GUI is created which displays the specific data according to the user’s requirement.
This is a very miniature version of how ATC works.
